# 2.7 How Much — Quanto custa?

## Resumo Executivo

**Custo de desenvolvimento: R$ 0,00 em ferramentas.**
**Custo de execução nas escolas: R$ 0,00 em licenças.**

Todo o stack tecnológico é open-source e gratuito. O único custo real é o **tempo da equipe** de desenvolvimento.

---

## Custo de Desenvolvimento

| Item | Custo | Observação |
| ---- | ----- | ---------- |
| Linguagem C + GCC/MinGW | R$ 0,00 | Open-source (licença GPL) |
| Raylib 5.x | R$ 0,00 | Open-source (licença zlib) |
| miniaudio (áudio) | R$ 0,00 | Embutido no Raylib (licença MIT) |
| VS Code / editor de texto | R$ 0,00 | Open-source |

| **Horas de desenvolvimento (equipe)** | ~600–800 h | 3 integrantes ao longo do semestre |

## Custo de Distribuição e Uso nas Escolas

| Item | Custo por máquina | Observação |
| ---- | ----------------- | ---------- |
| Licença do jogo | R$ 0,00 | MIT — uso e distribuição livres |
| Instalação (Raylib + executável) | R$ 0,00 | ~10 min por máquina, uma única vez |
| Espaço em disco | ~100 MB | Compatível com qualquer PC escolar moderno |
| Internet durante uso | Nenhuma | 100% offline |
| Atualização / manutenção | R$ 0,00 | GitHub — download direto da nova versão |

> **Em um laboratório de 30 computadores:** custo de implantação = R$ 0,00.
> Única exigência: hardware com OpenGL 3.3 e 512 MB de RAM — compatível com PCs escolares desde 2010.
